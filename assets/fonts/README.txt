中文字体目录
=============

为确保游戏在所有平台都能正确显示中文，可以在打包时将字体文件放在此目录：

Windows:
  - 复制 C:\Windows\Fonts\microsoftyahei.ttf (微软雅黑)
  - 或 simhei.ttf (黑体)

macOS:
  - 复制 /System/Library/Fonts/PingFang.ttc (苹方)
  - 或 /Library/Fonts/Microsoft/SimHei.ttf

Linux:
  - 安装并复制: sudo apt install fonts-wqy-microhei
  - 字体路径: /usr/share/fonts/truetype/wqy/wqy-microhei.ttc
  - 或从 https://github.com/googlefonts/noto-cjk 下载 NotoSansCJK

推荐字体（免费，支持中日韩）:
  - NotoSansCJK-Regular.ttc (Google + Adobe)
  - 下载地址: https://github.com/googlefonts/noto-cjk/releases
